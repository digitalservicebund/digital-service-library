# Button

`<Button>` is a drop-in replacement for `<button>`. You can use all the properties that you are used to. Only exception is `className`: when you set `className`, your value will be appended to the existing `className` of `<Button>`. This is handy to add margins, for example. In addition you can use the properties from the table below to configure the visual appearance. Use either `secondary` or `tertiary`, not both. Use either `small` or `tiny`, not both.

<ArgsTable of={Button} />

<Canvas>
  <Story
    name="Example"
    args={{
      small: true,
      type: "submit",
      className: "mb-2",
      onClick: () => alert('clicked'),
      children: "Example"
    }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Links which look like buttons

The component is overloaded. If you pass a `href` property and optionally other properties from `<a>`, the `<Button>` component will render an `<a>` link and not a `<button>`. The important piece is, that you pass a `href` property. Obviously you can then only use properties from `<a>`.

<Canvas>
  <Story
    name="Link"
    args={{
      href: "https://digitalservice.bund.de",
      target: "_blank",
      rel: "noopener",
      children: "This is a link"
    }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Edge cases

### Long text

Given a small device and a long button text it might happen that the text does not fit into one line.

<Canvas>
  <Story
    name="Long text"
    args={{
      children: "Der DigitalService4Germany entwickelt digitale Lösungen für und mit der Bundesverwaltung. Im Zentrum stehen die Bedürfnisse der Nutzerinnen und Nutzer. Unser Ziel: Dienstleistungen, die besser für alle funktionieren."
    }}>
    {Template.bind({})}
  </Story>
</Canvas>

### Long word

Some German words are really long and might not fit in one line on a small device. The word should be broken in several lines even if the breaking point is not optimal.

<Canvas>
  <Story
    name="Long word"
    args={{
      children: "DerDigitalService4GermanyentwickeltdigitaleLösungenfürundmitderBundesverwaltung.ImZentrumstehendieBedürfnissederNutzerinnenundNutzer.UnserZiel:Dienstleistungen,diebesserfürallefunktionieren."
    }}>
    {Template.bind({})}
  </Story>
</Canvas>
