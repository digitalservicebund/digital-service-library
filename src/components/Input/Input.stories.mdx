# Input

[Source code](https://github.com/digitalservice4germany/digital-service-library/tree/main/src/components/Input) | [Figma](https://www.figma.com/file/49Gxm2IajuwbGhj37zu9x0/Angie?node-id=0%3A1)

`<Input>` is a drop-in replacement for `<input>`. You can use all the properties that you are used to. Only exception is `className`: when you set `className`, your defaultValue will be appended to the existing `className` of `<Input>`. This is handy to add margins, for example. In addition you can use the properties from the table below to configure the visual appearance. Use either `small` or `tiny`, not both.
